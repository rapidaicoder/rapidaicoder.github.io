# Technology Stack

## Frontend & Content
- **Framework/Generator:** [Jekyll](https://jekyllrb.com/) (Standard for GitHub Pages)
- **Content Format:** [Markdown](https://daringfireball.net/projects/markdown/)
- **Templating:** Liquid (Jekyll default)
- **Styling:** CSS (Custom or Jekyll Theme-based)

## Infrastructure
- **Hosting:** [GitHub Pages](https://pages.github.com/)
- **Version Control:** Git & GitHub
- **CI/CD:** GitHub Actions (Automatic deployment to Pages)

## Client-Side Integration
- **GitHub Repository:** [rapidai/aicoder](https://github.com/rapidai/aicoder)
